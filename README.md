## 4.solo Sensors

以下内容源自：[这里-Multiparameter probe CTD 48M](https://www.sea-sun-tech.com/product/multiparameter-probe-ctd-48-memory/#detail)

Standard sensors:

- Conductivity (С)
- Temperature (T)
- Pressure (D)

Additional sensors:

- Dissolved Oxygen
- Turbidity
- pH
- Redox (OPR)

Calculated parameters:

- salinity
- sound velocity
